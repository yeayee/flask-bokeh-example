# Flask Bokeh 交互

## 假定Flask Web开发（狗书）

## 如何运行？

1. 下载代码
1. CMD路径切换至下载文件夹中
1. 运行`python app.py`
1. 在浏览器访问127.0.0.1

1. 在Bokeh1.0版本后，与Flask交互更好，极客必备。
